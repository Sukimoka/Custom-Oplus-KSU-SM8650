用于构建用于Oplus6.1系内核的类原生内核                      
1.为基于LineageOS DT构建内核请选择Lineage为Kernel和KernelModule，内核分支输入lineage-23.2                
2.为Crdroid构建请选择crdroidandroid为Kernel和KernelModule，内核分支输入16.2                        
3.为YAAP构建，请选择SukiMoka作为Kernel和KernelModule，分支输入sixteen                                     

注意：                                                                          
自定义内核名仅支持英文和字母及部分符号                      
请不要在内核名前加-，请直接输入你想表达的名字
